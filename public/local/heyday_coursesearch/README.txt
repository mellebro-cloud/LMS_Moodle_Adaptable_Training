Heyday Course Search local plugin v1.0.6
========================================

Install folder:
C:\xampp\htdocs\moodle\local\heyday_coursesearch

Test URL:
http://localhost/moodle/local/heyday_coursesearch/search.php?courseid=105

This version fixes the search-page top utility menu so it matches the ed2go reference more closely:
- one Search icon
- one Help icon
- Moodle user account menu
- no duplicate Help Center icon
- no Tour icon/text on the search page
- no notification/message icons on the search page

It also keeps the large learner-facing course search box in the page body.

Keep your Additional HTML course-search redirect script if you want course pages to open this plugin when clicking the Moodle search icon. Remove old custom topbar scripts if they conflict.
